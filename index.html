Ruby versions changelog

Ruby 3.2.x

3.2.0 (Dec 2022) - 3.2.2 (March 2023).

Status: Supported

Implementations: MRI 3.2.2

  • Internals WASI based WebAssembly support
  • Internals Production-ready YJIT
  • Internals Regexp improvements against ReDoS
  • Internals IRB improvements (SyntaxSuggest, ErrorHighlight)
  • Internals Bundler 2.4 now uses PubGrub resolver instead of Molinillo

Ruby 3.1.x

3.1.0 (Dec 2020) - 3.1.4 (March 2023).

Status: Supported

Implementations: MRI 3.1.4

  • Internals YJIT: New experimental in-process JIT compiler
  • Added debug gem: A new debugger
  • Added error_highlight: Fine-grained error location in backtrace
  • Added IRB Autocomplete and Documentation Display
  • Changed Values in Hash literals and keyword arguments can be omitted.
  • Changed Updates in RBS / TypeProf

Ruby 3.0.x

3.0.0 (Dec 2020) - 3.0.6 (March 2023).

Status: Security maintenance phase

Implementations: MRI 3.0.6

  • Internals MJIT improvements
  • Added Ractor - Actor-model like concurrent abstraction
  • Added Fiber Scheduler for light-weight concurrency
  • Added RBS - a language to describe the structure of Ruby classes
  • Added TypeProf - type analysis tool, generates RBS type signatures
  • Added Hash#except
  • Changed Keyword arguments are separated from other arguments
  • Changed Rightward assignment statement
  • Changed Reversed (again) order of backtrace
  • Changed Endless method definition def square(x) = x * x

Ruby 2.7.x

2.7.0 (Dec 2019) - 2.7.8 (March 2023).

Status: All maintenance endded on March 31 2023

Implementations: MRI 2.7.7

  • Added Pattern Matching
  • Changed REPL improvements
  • Changed Separation of positional and keyword arguments
  • Changed Argument forwarding
  • Internals Compaction GC

Ruby 2.6.x

2.6.0 (Dec 2018) - 2.6.10 (Apr 2022).

Status: All maintenance ended on April 12, 2022

Implementations: MRI 2.6.10

  • Added endless ranges (1..)
  • Added Enumerable#chain
  • Added function composition operators <<, >> to Proc
  • Changed #then alias for #yield_self
  • Internals JIT compiler --jit
  • Internals Bundler is installed as a default gem

Ruby 2.5.x

2.5.0 (Dec 2017) - 2.5.9 (Apr 2021).

Status: All maintenance ended on April 5, 2021

Implementations: MRI 2.5.9 JRuby 9.2.0.0

  • Added yield_self
  • Added rescue in blocks
  • Added Hash#slice and Hash#transform_keys
  • Added ERB#result_with_hash
  • Changed New Array aliases - #prepend (#unshift) and #append (#push)
  • Changed Pattern argument on Enumerable methods
  • Changed Keyword arguments on Struct.new
  • Changed Removed Top-level constant lookup
  • Changed Reverse order of backtrace (oldest call first)

Ruby 2.4.x

2.4.0 (Dec 2016) - 2.4.10 (Mar 2020).

Status: All maintenance ended on March 31, 2020

Implementations: MRI 2.4.10

  • Added Enumerable#sum
  • Added Integer#digits
  • Added Regexp#match?, Regexp#named_captures
  • Changed New constructor Logger.new(STDOUT, level:, progname:)
  • Changed Float: #round, #ceil, #floor, and #truncate now accept a precision
  • Removed Removed Fixnum and Bignum classes (unified to Integer)

Ruby 2.3.x

2.3.0 (Dec 2015) - 2.3.8 (Oct 2018).

Status: All maintenance ended on March 31, 2019

Implementations: MRI 2.3.8 JRuby 9.1.17.0

  • Added Safe navigation operator &.
  • Added Frozen string literals
  • Added Array#dig
  • Added Hash#fetch_values, Hash#to_proc, Hash#dig
  • Added 'Did you mean?'

Ruby 2.2.x

2.2.0 (Dec 2014) - 2.2.10 (March 2018).

Status: All maintenance ended on March 31, 2018

  • Removed callcc is obsolete - use Fiber instead
  • Removed DL has been removed from stdlib - use Fiddle instead
  • Internals Incremental GC - fixes performance for 'old' objects
  • Internals The introduction of GC for symbols (mortal/immortal)

Ruby 2.1.x

  • Added Generational GC - young / old objects
